Uppgift

Element

Ett element har en starttag, någon typ av innehåll och en sluttag. Varje element kan ha HTML-attribut specificerade. Element kan också ha innehåll, inklusive andra element och text.

HTML

HTML står för HyperText Markup Language. I kort betyder det att HTML är ett märkspråk, även kallat sidbeskrivningsspråk

ID

ID används för att definera ett ID som måste är unikt och de används till att identivfiera element när man linkar

Class

Class fungerar ungefär som ID men classer kan användas till flera element du kan lätt ändra css med hjälp av class genom att använda .classnamn i css filen, detta är användbart om man lätt vill ändra css på många element

Meta

Meta används av webbläsaren för att de ska veta hur de ska visa upp content tex. meta visa inte upp på sidan

Responsive

Responsive hemsidor kan "resizea" innehållet för att passa i fönstret eller på telefonen om en sida är responsive så anpassar den alltså sidans utseende för att få plats med all sidans inehåll på skärmen

HTTP

HTTP eller hypertext transfer protocol, standardprotokollet för att överföra hypertext på webben

HTTPs

Hypertext Transfer Protocol Secure är ett protokoll för krypterad transport av data för HTTP-protokollet. så basically samma som http fast krypterad för säkerhetsskull.

Compressed data

Om en fil comprimeras kan vara att man tar bort alla till exempel treor och sen när man de:compressar filen igen vet den att alla treor ska tilbaks. man gör detta för att göra filen mindre för att transportera

CDN

Ett innehållsleveransnätverk (CDN) är en geografiskt distribuerad grupp av servrar som arbetar tillsammans för att ge snabb leverans av internetinnehåll. En CDN gör snabb överföring av tillgångar som behövs för att ladda internetinnehåll inklusive HTML-sidor, bilder och videor. majoriteten av webbtrafiken görs idag via CDN.

Local Bootstrap

Den lokala bootstrap-metoden drar bootstrap-exempel från ett område för varje datapunkt. Till skillnad från den vanliga bootstrap är den särskilt användbar för heteroscedastisk data.

JQuery

jQuery is a lightweight, "write less, do more", JavaScript library (w3schools) gjorde för att förenkla att använda java-script på hemsidor, JQuery tar ett flertal linjer kod och gör om det till en rad så det blirr lättare att skriva koden

Flexbox

Flexbox Layouten är en lösning för ett mer effektivt sätt att lägga ut, justera och fördela utrymme mellan föremål i en container, även om deras storlek är okänd och / eller dynamisk därav ordet ”flex”.

Grid

CSS Grid Layout Module erbjuder ett rutnätbaserat layoutsystem med rader och kolumner, vilket gör det lättare att designa webbsidor utan att behöva använda flottör och positionering.

Code snippets

code snippets är formellt definierade operativa enheter som ska införlivas i större programmeringsmoduler. Utdragshantering är en funktion av vissa textredigerare, programkällkodredigerare, IDE och relaterad programvara. Det gör det möjligt för användaren att undvika upprepad skrivning under rutinmässiga redigeringsåtgärder.
